# Security Baselines Guide

## Purpose

Security baselines provide a recommended configuration to improve endpoint security and reduce attack surface.

## Recommended Controls

### Identity Protection

- Enable MFA
- Use Conditional Access
- Implement Privileged Identity Management (PIM)
- Maintain break-glass accounts

### Device Protection

- Enable BitLocker
- Configure Windows LAPS
- Enable Microsoft Defender for Endpoint using EDR
- Configure Windows 10/11 Security Baselines
- Configure Tamper Protection

### Attack Surface Reduction

- Enable ASR rules
- Block Office macros from the Internet
- Restrict executable content from user profile locations
- Monitor security events

### Compliance

- Require encryption
- Require antivirus protection
- Require secure boot
- Require supported operating systems

## Monitoring

- Review Secure Score
- Review compliance reports
- Review Defender recommendations
- Investigate high-risk devices
